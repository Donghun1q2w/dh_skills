# dh-dev 스킬 추가 — 코드 기능개선 오케스트레이터

- **Date**: 2026-03-16 15:00:00
- **Status**: Completed

## Summary

기존 코드의 기능 개선에 특화된 end-to-end 오케스트레이터 스킬. 코드 분석 → plan-context(기획) → 사용자 리뷰(승인/거부/코멘트) → ultrawork(구현) → revision-tracker(리뷰·커밋) 4단계를 하나의 워크플로우로 연쇄 실행한다.

## Background

소규모 코드 개선 작업(기능 추가, 리팩토링, 버그 수정, 성능 최적화 등)은 매번 동일한 흐름을 거침. 이 반복 흐름을 하나의 스킬로 자동화하여 일관성과 추적성을 확보하기 위해 dh-dev 스킬을 신규 추가한다.

## Proposal

1. `skills/dh-dev/SKILL.md` 생성 — 4단계 워크플로우 정의
2. Step 1: Analyze & Plan (explore + plan-context)
3. Step 2: User Review (승인/거부/코멘트 3선택지)
4. Step 3: Execute (ultrawork)
5. Step 4: Review & Commit (revision-tracker)
