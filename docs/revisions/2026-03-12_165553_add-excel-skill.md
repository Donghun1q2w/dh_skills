### `skills/excel/SKILL.md` (Added)

- description: Windows/macOS 크로스플랫폼, UTF-8 우선, 외부 의존성 없음 명시
- Encoding 섹션: UTF-8 → utf-8-sig → cp949 → euc-kr 순서 fallback 함수 제공, 파일 형식별 인코딩 처리 테이블

- 수식 오류 검증: openpyxl `data_only=True` 기반 Python 검증 코드 제공
